# PhotoBaby 📸👶

Proyecto final de recuperación inspirado en fotografía familiar y de bebés.

## Descripción

PhotoBaby es una página web desarrollada en HTML y CSS enfocada en mostrar un diseño moderno, limpio y responsive para una galería de fotografía de bebés y familias.

El proyecto ha sido creado como trabajo final de curso utilizando técnicas de maquetación responsive con **Flexbox** y **CSS Grid**.

---

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- Diseño Responsive

---

## Características

- Diseño adaptable a móviles, tablets y ordenador
- Galería de imágenes responsive
- Estructura moderna y sencilla
- Uso de Grid para la distribución principal
- Uso de Flexbox para alineación y organización de elementos

---

## Objetivos del proyecto

- Practicar maquetación web responsive
- Aplicar Flexbox y Grid en un proyecto real
- Mejorar la estructura y organización del código HTML y CSS
- Crear una web visual y atractiva
